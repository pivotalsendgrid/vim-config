# This is a place for me to keep notes about using vim

* <C-w> + j   --    Move to window below
* <C-w> + k   --    Move to window above
* <C-V>       --    Line edit mode

Verbs
------
* v   --  visual
* c   --  change
* d   --  delete
* y   --  yank

Location modifiers
-------------------
* i   --    inside
* t   --    until
* a   --    around
* f   --    find
* /   --    search

Objects
-------
* w   --  word
* s   --  sentence
* p   --  paragraph
* b   --  block/parentheses
* t   --  tag (ie. html)
